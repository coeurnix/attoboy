# Attoboy

Библиотека C++ с полной комплектацией для создания небольших утилит под Windows. Пишите чистый, читаемый код, который компилируется в исполняемые файлы размером менее 100 КБ.

```cpp
#include <attoboy/attoboy.h>
using namespace attoboy;

void atto_main() {
  Console con;
  String name = con.input("What's your name? ");
  con.println("Hello, " + name + "!", CON_CYAN);
  Exit();
}
```

Компилируется примерно в ~12 КБ. Никаких установщиков, никаких зависимостей, запускается на любой машине с Windows.

## Почему Attoboy?

Большинство C++ программ тянет за собой STL и CRT, что добавляет сотни килобайт накладных расходов. attoboy обходит всё это и работает напрямую с Windows API, предоставляя удобный интерфейс без лишнего веса.

Он разработан для программистов, которые только начинают работать с C++, а также для тех, кто создаёт консольные утилиты, файловые инструменты, веб-скрейперы и другие небольшие программы, где важны переносимость и размер.

## Что включено?

**Основные типы:**
- `String` — UTF-8 строки с методами в стиле Python (`split`, `replace`, `trim`, `contains` и т.д.)
- `List` — динамические массивы, допускающие смешанные типы, с поддержкой JSON/CSV
- `Map` — словари ключ-значение с гибкостью типов
- `Set` — коллекции уникальных значений с операциями над множествами

**Файловый ввод/вывод:**
- `Path` — метаданные файлов, навигация и простое чтение/запись
- `File` — потоковый ввод/вывод для файлов и сокетов
- `Buffer` — бинарные данные с поддержкой сжатия LZ4 и шифрования ChaCha20

**Сеть:**
- `WebRequest` / `WebResponse` — HTTP-клиент с поддержкой JSON
- `AI` / `Conversation` / `Embedding` — интеграция с API, совместимым с OpenAI

**Консольный интерфейс:**
- `Console` — цвета, меню, индикаторы прогресса, ввод с историей и автодополнением по Tab
- `Arguments` — парсинг командной строки с флагами и параметрами

**Система:**
- `DateTime` — метки времени с точностью до миллисекунды
- `Thread` / `Mutex` — лёгкие потоки
- `Subprocess` — запуск внешних программ с потоковым обменом ввода/вывода
- `Registry` — доступ к реестру Windows
- `Math` — стандартные математические функции и генерация случайных чисел

Все типы используют RAII, поэтому вам не нужно вручную управлять памятью. Строки неизменяемы. Всё потокобезопасно.

## Требования

- **Windows** (32-битные исполняемые файлы, работают во всех версиях)
- **Компилятор:** MSVC (рекомендуется) или MinGW GCC
- **Система сборки:** CMake 3.15+
- **Тестирование (необязательно):** Python 3.10+
